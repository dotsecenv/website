*Last updated: April 11, 2026*

## The short version

The dotsecenv CLI collects no data. This website collects no data. We don't use cookies, analytics, or tracking.

## CLI tool

The dotsecenv CLI:

- **Does not phone home.** No telemetry, no update checks, no network calls of any kind. This is verified by [hermetic testing](/concepts/security-model/#hermetic-testing) with network access disabled.
- **Does not collect or transmit your secrets.** All encryption and decryption happens locally using your GPG keys.
- **Stores data only on your filesystem.** Vault files, configuration, and keys never leave your machine unless you explicitly push them to a git remote.

## Website

This website (dotsecenv.com):

- Does not use cookies
- Does not use analytics or tracking scripts
- Does not collect personal information
- Is a static site hosted on GitHub Pages

## Third-party services

- **GitHub** hosts our source code and website. GitHub's own privacy policy applies when you visit our GitHub repositories. See [GitHub's Privacy Statement](https://docs.github.com/en/site-policy/privacy-policies/github-general-privacy-statement).
- **Package repositories** (Homebrew, APT, YUM, Arch) are used for distribution. Each has its own privacy policy.

## Contact

If you have questions about this policy, open an issue on [GitHub](https://github.com/dotsecenv/dotsecenv).
